# Bill Splitter CLI

A streamlined, interactive Command Line Interface (CLI) application built in Python to accurately calculate total bill breakdowns, including custom variable tax margins, tips, and equal division among a party.

## Features
* **Float Type Conversion Input:** Handles multi-variable user inputs dynamically, converting terminal string captures into floats and integers for precise math calculations.
* **Precise Arithmetic Operations:** Runs sequential calculation formulas to isolate tax amounts, calculate percentage tips on base totals, and map the comprehensive balance into an even split array.
* **Floating-Point Formatting:** Implements string interpolation templates with explicit financial rounding limits (`:.2f`) to ensure printed outputs match professional currency standards.
